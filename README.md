## Getting Started

### Prerequisites

- web browser (e.g. Google Chrome, Firefox)
- text editor (e.g. VSCode)
- CLI/terminal

### Clone this repository

```
git clone git@github.com:virag-ky/Weather-App.git
```

### Install dependencies

```
npm install
```

### Open the project in the browser

```
npm run start
```
